01. What is DOM and what is the purpose of DOM?

➡️The full name of DOM is Document Object Model. When a web interface is loaded in a browser, the browser represents the web interface in a DOM. DOM objects have a type tree and each type has an element of the web interface. Developers use JavaScript in the DOM to make web interfaces attractive and dynamic.

02. How will you select HTML elements using DOM? Name the DOM methods.

➡️DOM has certain rules for selecting HTML tags or elements. HTML tag, class name, id, css style, inner text, inner html can also be selected.

Some of the most commonly used methods are:

getElementsByTagName()
getElementsByClassName()
getElementById()
querySelector()
querySelectorAll()

03. What is event bubble?

➡️Event bubbles are a mechanism for propagating events in the HTML DOM. And it is managed by their parent elements before being managed by their child elements.
When an event occurs, the innermost element's handler runs, followed by the parent and then the next ancestor elements. In other words, events bubble or propagate up the DOM tree.

04. What is a callback function and why will you use it?

➡️A callback function is one of the great strengths of JavaScript. This is the JavaScript way of passing one function as an argument to another function. Arguments in JavaScript are values that are passed as parameters to a function. The functions that are called are accessible only to the functions that are passed to them.
